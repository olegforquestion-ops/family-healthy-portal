\# Acceptance checklist



\## UX stage

\- all required screens exist

\- key user flows are covered

\- meal draft flow is understandable

\- family dashboard is clear

\- mobile layout is considered



\## Technical core

\- auth works

\- admin can create users

\- profile saves

\- norms calculate correctly

\- migrations run



\## Functional MVP

\- meal logging works

\- draft recalculation works

\- water logging works

\- workout logging works

\- goals work

\- dashboards show correct aggregations



\## Deployment

\- app runs on Ubuntu via Docker Compose

\- PostgreSQL persists data

\- migrations can be applied

\- app is reachable behind reverse proxy
