\# Stack decision



\## Selected stack

\- Next.js

\- TypeScript

\- Tailwind CSS

\- shadcn/ui

\- PostgreSQL

\- Prisma

\- Auth.js / NextAuth credentials provider

\- Recharts

\- Docker / Docker Compose

\- Nginx reverse proxy on Ubuntu



\## Why

\- fast MVP delivery

\- single codebase

\- strong support for admin panels, forms, dashboards

\- easy deployment to Ubuntu server

\- easy for Codex to work with in one repository



\## Constraints

\- no separate backend unless explicitly required

\- no unnecessary microservices

\- no Windows-specific dependencies
