\# Seed requirements



The project must include seed/reference data for MVP.



\## Required seed data

\- roles:

&#x20; - admin

&#x20; - user



\- profile goal types:

&#x20; - weight\_loss

&#x20; - gentle\_weight\_loss

&#x20; - maintenance

&#x20; - mass\_gain



\- meal types:

&#x20; - breakfast

&#x20; - lunch

&#x20; - dinner

&#x20; - snack



\- goal types:

&#x20; - target\_weight\_by\_date

&#x20; - workouts\_count\_by\_date

&#x20; - distance\_by\_date

&#x20; - steps\_by\_date

&#x20; - calorie\_norm\_days

&#x20; - water\_norm\_days



\- workout norms:

&#x20; - walking\_per\_hour

&#x20; - running\_per\_hour

&#x20; - strength\_training\_per\_session



\## Requirements

\- seed scripts must be idempotent if possible

\- seed execution instructions must be in README

\- seed data names should be human-readable in UI
